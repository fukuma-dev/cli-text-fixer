# LineFix — CLI出力の改行崩れを直すWebツール

## 概要
CLIツール（Claude CLI, ChatGPT, ターミナル）の出力をコピペすると、80文字等で強制改行されて文章が崩れる問題を解決する単一ページWebツール。

## 要件

### 機能
- 左側: テキスト入力エリア（ペースト用）
- 右側: 整形済みテキスト表示
- リアルタイム変換（入力と同時に右側に反映）
- コピーボタン（整形済みテキストをクリップボードにコピー）
- クリアボタン

### 整形ロジック
1. 空行 = 段落区切り → 維持
2. `- ` `* ` `数字. ` で始まる行 = リスト → 改行維持
3. `#` で始まる行 = 見出し → 改行維持
4. `` ``` `` で囲まれたブロック = コードブロック → そのまま維持（中身の改行も保持）
5. `|` で始まる行 = テーブル → 改行維持
6. `>` で始まる行 = 引用 → 改行維持
7. インデントされた行（4スペース/タブ）= コード → 改行維持
8. それ以外の連続する行 → 1行に結合（不要な改行を除去）

### デザイン
- **ダークモード**メインのミニマルUI
- レスポンシブ（モバイルでは上下レイアウト）
- フォント: モノスペース（入力側）、サンセリフ（出力側）
- カラー: ダーク背景(#1a1a2e) + アクセント(#e94560)
- アニメーション: コピー成功時のフィードバック

### 技術
- **単一HTMLファイル**（index.html のみ）
- CSS・JS全てインライン
- 外部依存なし
- GitHub Pages でホスティング
